# Tutorial 8: Entropic Gravity

This tutorial introduces the integration of Bianconi's "Gravity from Entropy" framework with the GMET contact geometry library.

## Overview

The `entropic-gravity` module implements a dynamical principle where gravity emerges from the interplay between two metrics:
1. **Spacetime Metric ($g$)**: The quantum operator defining geometry.
2. **Matter-Induced Metric ($G$)**: The classical state defined by matter fields.

The dynamics are generated by minimizing the **Quantum Relative Entropy Action**:
$$ S(G||g) = \int d^4x \sqrt{|g|} \text{Tr}[G(\ln G - \ln g)] $$

## Key Components

### 1. Two-Metric System

The core object is the `TwoMetricSystem`, which manages $g$ and $G$.

```javascript
const EntropicGravity = require('../src/entropic-gravity.js');

// 1. Define Spacetime Metric (g)
const g = EntropicGravity.StandardMetrics.schwarzschild(1.0);  // Mass M=1

// 2. Define Matter-Induced Metric (G)
const G = new EntropicGravity.MatterInducedMetric({
    // Scalar field φ(x)
    scalarField: x => Math.exp(-(x[1]**2 + x[2]**2 + x[3]**2)),
    
    // Vector potential A_μ(x)
    vectorPotential: x => [1/Math.sqrt(x[1]**2 + x[2]**2 + x[3]**2), 0, 0, 0]
});

// 3. Create System
const system = new EntropicGravity.TwoMetricSystem(g, G);
```

### 2. Relative Entropy Action

Compute the action $S(G||g)$ and its variations.

```javascript
const action = new EntropicGravity.RelativeEntropyAction(system);

// Local entropy density at point x
const density = action.localDensity([0, 5, Math.PI/2, 0]);
console.log('Local entropy density:', density);

// Entropic stress-energy tensor T_ent = δS/δg
const T_ent = action.variationWrtMetric([0, 5, Math.PI/2, 0]);
```

### 3. Emergent Cosmological Constant

In this framework, $\Lambda$ is not a fixed parameter but an emergent field $\Lambda_G$ arising from the trace of the metric ratio.

$$ \Lambda_G = \frac{1}{4} \text{Tr}(G \cdot g^{-1} - 4I) $$

```javascript
const lambda = new EntropicGravity.EmergentCosmologicalConstant(system);

const Lambda_val = lambda.localValue([0, 10, 0, 0]);
const type = lambda.classify([0, 10, 0, 0]); // 'de Sitter', 'anti-de Sitter', or 'Minkowski'
```

### 4. Integration with GMET

The entropic action serves as a potential in the contact Hamiltonian.

$$ H = H_{geo} + \alpha S(G||g) $$

where $H_{geo}$ is the standard geodesic Hamiltonian.

```javascript
const GMET = require('../src/index.js');

// Create Grand Contact Manifold
const M13 = new GMET.GrandContactManifold();

// Create Hamiltonian
const H = new EntropicGravity.EntropicGravityHamiltonian(M13, system, {
    mass: 1.0,
    entropicCoupling: 0.01  // Strength of entropic force
});

// Create initial point (in contact phase space)
const p0 = M13.physicalPoint(
    10, Math.PI/2, 0, // Position (r, θ, φ)
    0,                // Time
    0,                // Log-scale
    1.0,              // Entropy
    0, 0, 0.1,        // Momenta
    1.0,              // Frequency
    0,                // Dilatation
    1.0,              // Temperature
    0                 // Action
);

// Integrate trajectory
const trajectory = H.flow(p0, 0.1, 100); // 100 steps
```

## Example: Entropic Trajectory

The full simulation shows how the particle's path deviates from a standard geodesic due to the entropic force.

```javascript
// Compare final positions
const p_final = trajectory[trajectory.length - 1];
```

## Geometric Algebra Formulation

The module now supports a coordinate-free formulation using **Spacetime Geometric Algebra** (Cl(1,3)).

### Why GA?
Standard Entropic Gravity minimizes $S(G||g)$. In GA, we can express the "entropic potential" directly via the **Connection Bivector Field** $\omega_\mu$.

$$ H_{ent} \approx \alpha \langle \omega_\mu \omega^\mu \rangle $$

This treats gravity as a gauge field strength rather than a metric deformation.

### Usage

Enable GA mode in the Hamiltonian:

```javascript
const H_ga = new EntropicGravity.EntropicGravityHamiltonian(M13, system, {
    mass: 1.0,
    entropicCoupling: 0.01,
    useGA: true  // <--- Activates Cl(1,3) solver
});
```

This uses `SpacetimeManifoldGA` to compute tetrads and spin connections automatically from the metric.
